# MyBodyGuard.ai Image Specifications

## Required Images for Landing Page

### Priority 1: Must-Have Images (Launch Blockers)

#### 1. Hero Guardian Visual
**Purpose:** Main hero section centerpiece, first thing visitors see  
**Current:** Placeholder with emoji shield  
**Needed:** Professional guardian concept

**Specifications:**
- **Dimensions:** 800x800px minimum (square), 1200x1200px ideal
- **Format:** PNG with transparency (preferred) or high-quality JPG
- **File Size:** <500KB (optimize for web)
- **Style:** 
  - 3D rendered guardian character OR
  - Abstract shield/guardian concept OR
  - Animated holographic effect OR
  - Stylized robot/AI guardian
- **Colors:** Blues (#0066FF), cyans (#00C2FF), purples (#7B61FF)
- **Mood:** Protective, futuristic, trustworthy, powerful
- **Background:** Transparent or subtle gradient
- **File Name:** `hero-guardian.png`

**Design Notes:**
- Should feel premium, not cartoonish
- Avoid medical imagery (no crosses, stethoscopes)
- Think Iron Man's JARVIS meets modern security
- Could incorporate shield, fortress, or guardian angel concepts

**Where Used:**
- Hero section (main landing area)
- Potential use in social media cards
- Could be animated version for video/GIF

#### 2. Merchandise Hero Shot
**Purpose:** Show "THIS BODY is protected by ai" apparel in real context  
**Current:** Placeholder outline  
**Needed:** Professional lifestyle product photo

**Specifications:**
- **Dimensions:** 1200x1400px (portrait orientation)
- **Format:** JPG, high quality (80-90% compression)
- **File Size:** <800KB
- **Content:** Person wearing the shirt
  - Age: 25-40 years old
  - Setting: Urban/modern environment or clean studio
  - Lighting: Natural or professional studio lighting
  - Pose: Casual, confident, front-facing
  - Focus: Shirt text clearly readable
  - Crop: Chest to mid-thigh or full body
- **Shirt Details:**
  - Text clearly visible
  - Fabric looks premium (not cheap)
  - Color: Black, navy, or white (primary colorways)
- **File Name:** `merch-hero.jpg`

**Design Notes:**
- Should NOT look like a mockup/template
- Real photography > generated images
- Diverse representation preferred
- Avoid stock photo feel - should feel authentic
- Model should look empowered, not generic

**Where Used:**
- Movement section (main merchandise showcase)
- Potential use in ads/social media
- Email campaigns

## Image Optimization Checklist

### Before Upload
- [ ] Resize to exact specifications (don't upload huge files)
- [ ] Compress images (use TinyPNG, ImageOptim, or similar)
- [ ] Convert to correct format (PNG for transparency, JPG for photos)
- [ ] Ensure color profile is sRGB for web
- [ ] Test images load quickly on slow connections

### Tools Recommended
- **Compression:** TinyPNG.com, Squoosh.app
- **Resizing:** Photoshop, GIMP, Affinity Photo, Canva
- **Mockups:** Figma, Sketch, Adobe XD
- **Stock Photos:** Unsplash, Pexels (if needed for inspiration)
- **3D Rendering:** Blender, Cinema 4D, or Sloyd.ai (as discussed)

## Integration Instructions

### How to Add Images to HTML

#### Replace Hero Guardian:
```html
<!-- Find this div (around line 98): -->
<div class="guardian-placeholder">
    <!-- Delete all this -->
</div>

<!-- Replace with: -->
<img src="images/hero-guardian.png" 
     alt="MyBodyGuard AI Personal Guardian" 
     class="hero-guardian-image">
```

#### Replace Merchandise Photo:
```html
<!-- Find this div (around line 450): -->
<div class="merch-mock">
    <!-- Delete all this -->
</div>

<!-- Replace with: -->
<img src="images/merch-hero.jpg" 
     alt="THIS BODY is protected by ai merchandise" 
     class="merch-hero-image">
```

#### Add Styling for New Images:
Add to CSS file:
```css
.hero-guardian-image {
    width: 100%;
    max-width: 500px;
    height: auto;
    filter: drop-shadow(0 0 30px rgba(0, 102, 255, 0.5));
    animation: pulse 3s ease-in-out infinite;
}

.merch-hero-image {
    width: 100%;
    border-radius: 1rem;
    box-shadow: 0 20px 25px rgba(0, 0, 0, 0.2);
}
```

## Production Ready Checklist

### Image Quality
- [ ] All images optimized for web (<500KB each)
- [ ] Images are correct dimensions
- [ ] Images use web-safe color profiles (sRGB)
- [ ] No placeholder images remain
- [ ] All alt text is descriptive and accurate

### Visual Consistency
- [ ] Colors match brand (#0066FF blue theme)
- [ ] Style is consistent across all images
- [ ] Photography has similar lighting/quality
- [ ] Mockups look professional, not template-y

### Technical
- [ ] Images load quickly (<2 second page load)
- [ ] Images are responsive (look good on mobile)
- [ ] Retina/high-DPI versions provided where needed
- [ ] Lazy loading implemented for below-fold images
